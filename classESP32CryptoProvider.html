Detailed Description

Concrete CW_CryptoProvider for ESP32 using mbedTLS and the hardware RNG.

SHA-256 / SHA-512 — mbedTLS software (hardware-accelerated on ESP32-S3). AES-CBC — mbedTLS software (hardware-accelerated on ESP32-S3). ECDH / key-gen — uECC shim backed by mbedTLS ECP primitives. Random — ESP32 hardware TRNG via esp_fill_random(); requires Wi-Fi or BT active (SEC-001).
